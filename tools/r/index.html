  • ▪ Hız: Derlenen dillere (C++, Julia) kıyasla döngü (for loop) işlemleri daha yavaş çalışabilir (vektörizasyon tercih edilmelidir).

Ekonometrik Regresyon Analizi

Aşağıdaki örnekte, R dilinde popüler dplyr ve ggplot2 paketleri kullanılarak bir veri setinin manipüle edilmesi, doğrusal regresyon modelinin kurulması ve sonuçların görselleştirilmesi gösterilmektedir: 

# Gerekli kütüphanelerin yüklenmesi
library(tidyverse)
library(broom)

# Örnek veri setinin yüklenmesi (mtcars)
data <- mtcars %>%
  select(mpg, wt, hp) %>%
  filter(hp > 100)

# Çoklu Doğrusal Regresyon Modelinin Kurulması
# Bağımlı Değişken: mpg (Mil/Galon), Bağımsız Değişkenler: wt (Ağırlık) ve hp (Beygir Gücü)
model <- lm(mpg ~ wt + hp, data = data)

# Model özet çıktısının düzenli tablo halinde alınması
summary_table <- tidy(model)
print(summary_table)

# Regresyon doğrusu ile saçılım grafiği çizimi
ggplot(data, aes(x = wt, y = mpg)) +
  geom_point(color = "#38bdf8", size = 3) +
  geom_smooth(method = "lm", color = "#fbbf24", se = TRUE) +
  labs(title = "Araç Ağırlığı ve Yakıt Tüketimi İlişkisi",
       x = "Ağırlık (1000 lbs)", y = "Mil / Galon (mpg)") +
  theme_minimal()
