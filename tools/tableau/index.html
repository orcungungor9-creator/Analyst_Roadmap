💻 Tableau LOD & Formül Örneği

LOD Expressions (FIXED, INCLUDE, EXCLUDE)

Aşağıdaki örnekte, Tableau'nun en güçlü analitik yeteneği olan LOD (Level of Detail) hesaplamaları ile müşteri başına ilk sipariş tarihinin bulunması, sepetteki ortalama ürün sayısının hesaplanması ve mantıksal KPI gruplamaları gösterilmektedir: 

// 1. FIXED LOD: Müşterinin Yaptığı İlk Sipariş Tarihi (Müşteri Kazanım Tarihi)
// Grafikteki zaman kırılımından bağımsız olarak her müşterinin en eski tarihini sabitler.
{ FIXED [Customer ID] : MIN([Order Date]) }


// 2. INCLUDE LOD: Sipariş Başına Ortalama Satış Tutarı
// Görselde sadece "Bölge" bazında analiz yapsanız bile, hesaplamaya "Sipariş ID" kırılımını dahil eder.
AVG( { INCLUDE [Order ID] : SUM([Sales]) } )


// 3. EXCLUDE LOD: Bölge Toplamına Oran (Kategori Bazında Yüzde)
// Görseldeki "Alt Kategori" kırılımını yok sayarak Bölge bazındaki toplam satışı bulur ve böler.
SUM([Sales]) / SUM( { EXCLUDE [Sub-Category] : SUM([Sales]) } )


// 4. Mantıksal Koşul (IF / ELSE): Kar Marjına Göre Dinamik Renklendirme ve Gruplama
IF SUM([Profit]) / SUM([Sales]) > 0.20 THEN "Yüksek Karlılık (Yeşil)"
ELSEIF SUM([Profit]) / SUM([Sales]) >= 0 THEN "Normal Karlılık (Sarı)"
ELSE "Zarar / Riskli (Kırmızı)"
END
